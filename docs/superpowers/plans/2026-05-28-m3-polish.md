# M3 Polish Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Close M3-era UX gaps that surface during normal use before starting M4. Scope is intentionally narrow — capture-path bugs and visibility, not new features.

**Architecture:** The popup currently relies on transient React state (`saveState`) that dies when the popup closes. M3 already persists everything needed in IndexedDB; this plan exposes that state on popup mount so the user gets continuous feedback across popup open/close cycles. No worker logic changes for tagging or storage — only a new read path keyed by normalized URL.

**Tech Stack:** Existing — TypeScript strict, React 18, Dexie, Vitest, Testing Library.

---

## Scope

In:

- Popup queries the worker on mount with the current tab's URL and renders one of: not-saved, processing, saved (with timestamp), failed.
- Worker adds `page.statusForUrl` handler that normalizes the URL and looks up by `urlHash`.
- `PageRepo` adds `getByUrlHash` (used by the new handler).
- While popup is open and a page is `pending`, popup polls every 2s until terminal state.

Out (deferred):

- Faster content extraction (premature without profiling).
- Retry button for failed pages (M6).
- Re-save / re-tag UI (future).
- Persistent storage permission prompt (separate concern).

Baseline before plan creation:

- `pnpm test`: 123 passed across 36 files.
- `pnpm typecheck`: passes.
- Storage stats and PageCard expand/collapse already shipped (post-M3 polish, committed directly — not part of this plan).

- [ ] **Step 2: Add handler case**

```ts
case "page.statusForUrl": {
  const { urlHash } = await normalizeUrl(request.payload.url);
  const page = await deps.pageRepo.getByUrlHash(urlHash);

  if (!page) {
    return { type: "page.urlStatus", payload: { saved: false } };
  }

  return {
    type: "page.urlStatus",
    payload: {
      saved: true,
      status: page.status,
      savedAt: page.savedAt,
      ...(page.errorReason ? { errorReason: page.errorReason } : {}),
    },
  };
}
```

Import `normalizeUrl` from `../lib/urlNormalize`.
